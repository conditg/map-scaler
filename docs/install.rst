Installation
===============
    
Installing from PyPI
^^^^^^^^^^^^^^^^^^^^^^^^^
``pip install mapscaler``

Dependencies
^^^^^^^^^^^^
* `NumPy <https://numpy.org/>`_
* `GeoPandas <https://geopandas.org/index.html>`_
* `Shapely <https://shapely.readthedocs.io/en/latest/index.html>`_
